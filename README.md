## Features

- Authentication with NextAuth.js
- Realtime messaging
- Adding friends and sending friend requests
- Protections of routes
- Responsive UI
- Dark mode
- Global chat
- Groups

## Built with

- Next.js 14
- Typescript
- Tailwind CSS
- Shadcn-ui
- Drizzle ORM
